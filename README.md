## 😎 Features

- 🍉 界面简洁，功能强大，资源占用少
- 🍊 上手简单，可视化操作，可0代码使用
- 🍠 一键切换内、外网模式链接
- 🍵 支持docker部署（支持Arm系统）
- 🎪 支持多账号隔离使用
